Replace Any Image or Video

Click Replace next to any item, pick your new file, then click Download. The file downloads with the exact correct filename — just move it into the matching folder (images/ or videos/) and refresh the site.

How to use: Click Replace → pick your new file → click the gold "Download as [filename]" button → copy the downloaded file into the sitarboston-c/images/ or sitarboston-c/videos/ folder, overwriting the old file → refresh your browser tab.

Live Site Text Editor

Edit any text section of the website here. When done, click Save & Download index.html — the file downloads with your changes applied. Replace the existing index.html on your server.

How it works: This editor loads the live index.html, lets you edit every major text section, then generates and downloads a complete updated file. No manual copy-pasting or HTML editing needed.

Note: This works when both admin.html and index.html are on the same server or opened locally from the same folder. If sections show as empty, click "Reload from File" below.

Tip: HTML tags inside text areas are preserved — you can use <strong>, <em>, <br> etc. for formatting. Do not add entire block elements like <p> or <div>.
